## 二、创建项目目录结构

| 目录       | 描述                                        |
| ---------- | ------------------------------------------- |
| assets     | 存放静态资源.比如: 图片、样式文件、字体文件 |
| config     | 存在配置文件                                |
| model      | 存放模型文件, 比如 api封装                  |
| pages      | 存放页面                                    |
| components | 存放组件                                    |
| utils      | 存放工具类方法文件                          |
| app.js     | 小程序全局入口文件                          |
| app.json   | 小程序全局配置文件                          |
